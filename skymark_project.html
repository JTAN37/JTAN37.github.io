Project Description

Skymarkprops.com is a website that I made for a real estate company based in Houston, Texas. The website displays the land properties that they are selling in Freeport, Texas. Each property has a web page that describes the property's size, location, land plot, and bordering roads.

The front end of the website was made using React and was designed with Bootstrap. For the back end, I used Node.js and deployed the website using GoDaddy. I used the React Leaflet API to create Leaflet maps for each property.
